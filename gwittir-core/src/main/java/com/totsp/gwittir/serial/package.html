This packages contains a series of CODECs that can be autogenerated for @Introspectable types for use in coms layers. Any types serialized in this fashion will be constructed using a no-args constructor, and their values set through the appropriate getters and setters, allowing for continutity work to be done in that code. For example: 


public interface TestBeanCodec extends JSONCodec<TestBean> {

}

...

 TestBeanCodec codec = GWT.create(TestBeanCodec.class);
 TestBean b = codec.deserialize(jsonString);
